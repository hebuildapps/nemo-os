# Nemo OS — Design Rules

## Visual Direction
- Retro / pixel-inspired aesthetic
- Fonts: Press Start 2P (pixel), IBM Plex Mono (body)
- Minimal, clean, cozy but practical
- Desktop-first with mobile adaptation

## Color Palette (HSL)
- Background: 40 20% 95%
- Surface: 0 0% 100%
- Foreground: 40 10% 10%
- Coin: 36 55% 50%
- Green: 140 25% 38%
- Red: 0 48% 36%
- Blue: 210 50% 36%

## Layout Rules
- 3-panel shell: character panel | workspace | icon rail
- No route-based page switching for workspaces
- Character panel persistent on desktop

## Avoid
- Glassmorphism, glossy fintech, cluttered dashboards, giant navbars
