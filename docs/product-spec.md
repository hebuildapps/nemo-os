# Nemo OS — Product Specification

## Product Summary
Nemo OS is a gamified placement preparation operating system for students preparing for software development interviews.

## Core Layout
- Left: Persistent character panel (260px)
- Center: Dynamic workspace
- Right: Icon rail (70px)

## Workspaces
- Calendar, Tasks, Badges, Shop, Profile

## Phase 1 (Current)
- Auth, onboarding, plan generation, calendar, tasks, MCQ validation, coins, badges, shop, profile, referral, adaptive replan placeholder

## Phase 2
- Stone progression display, tier system (Iron → Radiant)

## Phase 3 (Future)
- AI personalization, social systems, leaderboards, analytics, premium features
